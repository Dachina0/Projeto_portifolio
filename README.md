# Projeto_portifolio

## comandos github

`ìniciar repositório`
* git init
`carregar arquivos no stage`
* git add -A
`comentar as modificações`
* git commit -m ''
`upload para repositório`
* git push 
`ver status`
* git status
`ver alterações nos arquivos`
* git diff
`alteração de arquivo já enviado`
* git commit --amend
`forçar envio de modificações`
* git push --force
* git push -f
